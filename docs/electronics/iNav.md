## iNav
iNav stands for Intelligent Navigation, it is fork of betaflight with focused on adding gps navigation and to simplify the process of configuration of drones. Where Betaflight focuses on freestyle fpv drone operation with remote, iNav focuces on gps based navigation (autonomous navigation, waypoint navigation, position hold using gps and geozoning)

## iNav Configurator
It isa GUI software to configure drone with components like gyroscope, compass, gps, motors and flight controller.

## iNav Firmware
It is installed on flight controller, and does operations based on input received from navigational components. Firmware can be installed on to flight controller directly from iNav Configurator software.

## Geozones
iNav has introduced geozones in version 8.0.1. This helps drone pilot to mark out of bound areas or no-fly zones.
